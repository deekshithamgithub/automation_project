# automation_project

This project automates the login functionality of the OrangeHRM Demo application using Selenium WebDriver and TestNG.

## Tech Stack
- Java
- Selenium
- TestNG
- Maven
- GitHub

## Automated Test Cases Covered
Valid username and valid password
Invalid username and invalid password
Valid username and invalid password
Invalid username and valid password
Username with leading and trailing spaces
Password masking validation
Login button enablement validation

## Methodology
Agile Scrum (Milestone 1)
